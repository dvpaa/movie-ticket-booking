## 프로젝트 개요
COMET 방법론을 기반으로 하여, 자바의 객체지향적 특성을 활용함으로써 효율적이고 확장 가능한 영화 예매 시스템의 설계 및 구현에 초점을 맞춘다.

시스템 모델링에 중점을 두어, GUI와 데이터베이스 구성요소는 배제하고, 콘솔 기반의 인터페이스를 통해 구현하였다.
